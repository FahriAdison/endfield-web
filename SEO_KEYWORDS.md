# SEO Keywords Endfield Web

Dokumen ini jadi patokan keyword agar update konten lebih konsisten dan gampang dicari Google.

## Cara Membuat Keyword (Step by Step)

1. Tentukan intent pencarian:
- `Informasi`: contoh `apa itu arknights endfield`
- `Panduan`: contoh `build yvonne endfield`
- `Perbandingan`: contoh `tierlist endfield terbaru`

2. Pilih 1 topik utama per halaman:
- Satu halaman fokus ke satu cluster keyword.
- Hindari campur terlalu banyak intent dalam satu halaman.

3. Buat struktur keyword:
- `Head keyword` (umum): volume lebih besar, kompetitif.
- `Long-tail keyword` (spesifik): volume kecil, lebih mudah rank.

4. Terapkan keyword di tempat penting:
- `<title>`
- `meta description`
- `meta keywords`
- `<h1>` dan subjudul
- 1-2 paragraf awal konten

5. Perkuat internal link:
- Hubungkan halaman relevan (contoh: `/tierlist/` ke `/character/?id=...`).

6. Review berkala:
- Update keyword setiap ada patch/balance/banner baru.
